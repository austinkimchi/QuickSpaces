## Overview
**QuickSpaces** is a web application that provides a platform for commercial buyers' agents to browse leases in a particular province for a specific type of shop. <br/>
An AI algorithm gives a quick overview of the area based on safety, popularity, and walkability, powered by **NVIDIA Nemotron Nano 9b-v2 model**.
